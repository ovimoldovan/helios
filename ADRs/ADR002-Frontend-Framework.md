# ADR002 - Frontend Framework
___
## Status
Proposed

___
## Context
We need to choose a JS / TS framework for the frontend

___
## Decision
Use React for the frontend. Whether we use JS or TS is to be decided (TS should be the favourite here, all valid JS is valid TS anyway so it shouldn't matter).

___
## Consequences
- **Pros:** Huge ecosystem - libraries for nearly everything (routing, states, forms, animations), Half of the team already knows React, Easier to create a mobile interface if needed 
- **Cons:** More boilerplate

___
## Alternatives Considered
- Vue.JS (Technical overhead concerns)
